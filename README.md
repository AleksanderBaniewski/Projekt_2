# Projekt_2
Program przedstawia symulacje oczyszczalni sciekow.

---Sterowanie---
Cale sterowanie odbywa sie za pomoca przyciskow na panelu, ktory znajduje sie w lewym dolnym rogu okna
Mamy mozliwosc otworzenia zbiornika1, natychmiastowe napelnienie zbiornikow 2 lub 3, oproznienie zbiornika 4, reset symulacji, a takze dobor predkosci przeplywu cieczy za pomoca suwaka

---Dzialanie symulacji---
W pierwszym zbiorniku znajduje sie zielona zanieczyszczona woda, w zbiorniku 2 znajduje sie filtr ktory uruchamia sie po pelnym napelnieniu zbiornika, po zakonczeniu filtracji woda przelewa sie do zbiornika 3
ktory jest odpowiedzialny za jej podgrzanie, nastepnie woda trafia do 4 zbiornika, ktory mozemy oproznic, a woda z niego przeleje sie do docelowego odbiorcy. Procesy filtracji oraz nagrzewania wody widoczne sa na panelu sterowania. Woda zmienia kolor po kazdym procesie.
